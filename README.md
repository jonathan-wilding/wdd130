# WDD 130 - Web Design Fundamentals
BYU-Idaho Spring 2026

A collection of web design projects built while learning 
HTML and CSS fundamentals.

## Projects
- **Apostle Spotlight** - Transformed a wireframe concept into fully functioning HTML and CSS code.
  Enhanced the user experience by adding hover effects to the images.
- **Flags** - Explored using CSS Grid properties to configure different color layouts.
- **Gallery** - Experimented with photo editing software to reduce file sizes and optimize web page loading speeds.
- **Prophet Cards** - Discovered how to transform a wireframe into working HTML and CSS code.
  Challenged myself by adding a hover effect to the image.
- **Signup** - Practiced styling HTML elements and observed how they interact with JavaScript components.
- **CSS Positioning** - Practiced using CSS Grid alignment properties across various layouts.
- **Father's Day** - Focused on the CSS portion of the code and added hover effects to create a unique creative edge.
- **Favorite Devotional** - Gained experience creating forms and tables to display and collect data.
  Challenged myself by adding a feature to mark a devotional as an "All-Time Favorite."
- **White Water Rafting Site** - Developed a functioning website using basic HTML and CSS
